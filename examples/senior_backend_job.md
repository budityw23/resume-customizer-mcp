# Senior Backend Engineer - Financial Technology Platform

**Company:** TechFinance Solutions
**Location:** Remote (Indonesia/Southeast Asia)
**Employment Type:** Full-time
**Experience Level:** Senior (5+ years)

## About the Role

We are seeking an experienced Senior Backend Engineer to join our fintech platform team. You will be responsible for designing and implementing scalable microservices, optimizing system performance, and ensuring high availability for our financial products serving millions of users across Southeast Asia.

- Java Spring Boot (5+ years)
- Microservices Architecture
- RESTful API Design
- PostgreSQL or MySQL
- Docker and Kubernetes
- Event-driven architecture (RabbitMQ, Kafka)
- Cloud platforms (AWS or Google Cloud)

### Additional Technical Skills
- Domain-Driven Design
- CQRS Pattern
- Git version control
- CI/CD pipelines
- Monitoring and observability tools (Sentry, Datadog, etc.)

### Soft Skills
- Strong problem-solving abilities
- Excellent communication skills
- Experience mentoring junior developers
- Cross-functional collaboration
- Proactive and self-driven

## Preferred Qualifications

- Experience with financial/fintech products
- Knowledge of Angular or React
- Experience with message queues (RabbitMQ, Kafka)
- Background in high-traffic systems (1M+ users)
- Experience with NoSQL databases (MongoDB, Redis)
- Familiarity with Python for data processing
- Understanding of security best practices
- Experience with Agile/Scrum methodologies

## Responsibilities

- Design and develop scalable microservices using Java Spring Boot
- Implement event-driven architectures for real-time data processing
- Optimize database queries and system performance for high-traffic applications
- Collaborate with frontend developers to design robust APIs
- Ensure high availability and reliability of financial services
- Implement comprehensive testing strategies (unit, integration, e2e)
- Monitor system performance and troubleshoot production issues

- Participate in architectural decisions and technical planning
- Work with DevOps team to improve deployment processes
- Maintain high code quality and follow best practices
- Contribute to technical documentation
