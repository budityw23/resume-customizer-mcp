# Senior Full Stack Engineer - E-commerce Platform

**Company:** ShopFlow Technologies
**Location:** Hybrid - Jakarta, Indonesia
**Employment Type:** Full-time
**Experience Level:** Senior (6+ years)

## About the Role

Join our team to build the next generation of e-commerce solutions serving millions of customers across Indonesia. We're looking for a versatile Senior Full Stack Engineer who can work across the entire stack, from Angular frontends to Java Spring Boot backends, with a focus on building scalable, high-performance systems.

## Required Skills

### Frontend Skills
- Angular (4+ years) with NgRx and RxJS
- TypeScript
- Modern CSS (Flexbox, Grid, animations)
- Responsive design and mobile-first approach
- Angular Material or similar UI frameworks

### Backend Skills
- Java Spring Boot (4+ years)
- Node.js (NestJS) experience is a plus
- RESTful API design and implementation
- Database design (SQL and NoSQL)
- Microservices architecture

### DevOps & Tools
- Docker and container orchestration
- Git version control
- CI/CD pipelines
- Cloud platforms (AWS or GCP)
- Monitoring and logging tools

### Soft Skills
- Strong analytical and problem-solving skills
- Excellent communication and collaboration
- Experience working with product and design teams
- Ability to mentor junior developers
- Self-motivated and proactive

## Preferred Qualifications

- Experience with React (Context API, Hooks)
- Knowledge of RabbitMQ or similar message queues
- Experience with Elasticsearch or Solr
- Background in high-traffic systems (10M+ monthly users)
- Familiarity with Domain-Driven Design
- Experience with performance optimization
- Understanding of SEO and web vitals
- Experience with real-time features (WebSockets, Firebase)
- Knowledge of security best practices
- Experience with A/B testing and analytics

- Develop and maintain both frontend (Angular) and backend (Java Spring Boot) applications
